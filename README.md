> Program that generate critical case for image classification CNN model

### 🏠 [Homepage](https://github.com/myoons/cs454-critical-case-generator)

## Prerequisites

- python >=3.7.7
- pytorch >=1.6.0

## Install

```sh
install -r requirement.txt 
```

## Usage

```sh
python --model small main.py
```
